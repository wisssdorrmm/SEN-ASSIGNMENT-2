Full SDLC Explanation
1. Planning

Decide to create a personal portfolio webpage as the project.

Define the purpose: showcase student information, skills, and contact details.

Identify resources needed: VS Code, HTML, CSS, browser.

2. Requirement Analysis

Functional Requirements:

Display student information: name, department, matric number

Show sections for “About Me”, “Skills”, and “Contact”

Navigation within the page is clear and easy

Non-Functional Requirements:

Responsive design for different screen sizes

Clean and professional appearance

3. Design

Layout: Header → About Me → Skills → Contact → Footer

Styling: CSS for colors, spacing, fonts, and shadows

User Interface: Simple and easy-to-read sections

4. Implementation

Written in HTML5 for structure

CSS3 for styling

Organized the code with semantic HTML tags (header, section, footer)

Created separate style.css file for styles

5. Testing

Opened the webpage in multiple browsers (Chrome, Edge, Safari)

Checked that all text, sections, and formatting display correctly

Ensured that spacing, font sizes, and colors are consistent

6. Deployment

Project files can be deployed locally by opening index.html in a browser

Can also be uploaded to GitHub Pages for online access

7. Maintenance

Future updates can include:

Adding more sections like “Projects” or “Certificates”

Updating skills or contact details

Improving styling or adding interactive elements

Project Features

Header: Displays student name, department, and matric number

About Me: Short introduction about the student

Skills: Lists technical and personal skills

Contact: Provides email and phone contact details

Footer: Copyright and ownership information

Clean and professional design suitable for portfolio presentation

Tools Used

HTML5

CSS3

Visual Studio Code

How to Run

Download or clone the repository.

Open index.html in any modern web browser.

Navigate through the sections: About Me, Skills, and Contact.
